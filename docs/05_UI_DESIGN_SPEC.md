## 1. 전체 페이지 구조

```
/ (홈 — 랜딩 페이지)
│
├── /auth/login             로그인
├── /auth/register          회원가입
├── /auth/forgot-password   비밀번호 찾기
│
├── /dashboard              대시보드 (메인)
│
├── /accounts               계좌 연동 목록
│   └── /accounts/add       계좌 추가
│
├── /bots                   봇 목록
│   ├── /bots/new           봇 생성 (봇 타입 선택)
│   │   ├── /bots/new/spot-grid
│   │   ├── /bots/new/snowball
│   │   ├── /bots/new/rebalancing
│   │   ├── /bots/new/dca
│   │   └── /bots/new/algo-orders
│   └── /bots/:id           봇 상세 / 운영 현황
│
├── /settings               사용자 설정
│   ├── /settings/profile   프로필 설정
│   ├── /settings/security  보안 설정 (비밀번호, 2FA)
│   └── /settings/notifications  알림 설정
│
└── /billing                구독 / 결제
    ├── /billing/plans       플랜 선택
    └── /billing/history     결제 내역
```

## 2. 공통 레이아웃

### 2.1 네비게이션 (사이드바 + 상단 헤더)

```
┌─────────────────────────────────────────────────────────────┐
│  [로고]  AutoTrade                       [알림🔔] [프로필▼]  │  ← 상단 헤더
├──────────────┬──────────────────────────────────────────────┤
│              │                                              │
│  [🏠 대시보드 │                                              │
│              │                                              │
│  [🤖 봇 관리 │                                              │
│              │                 메인 컨텐츠 영역              │
│  [🔗 계좌 연동│                                              │
│              │                                              │
│  [💳 구독 관리│                                              │
│              │                                              │
│  [⚙️ 설정   │                                              │
│              │                                              │
└──────────────┴──────────────────────────────────────────────┘
```

### 2.2 공통 UX 컴포넌트

- **Toast 알림:** 우측 상단, 성공(초록)/실패(빨강)/정보(파랑)
- **Loading Skeleton:** 데이터 로딩 중 UI 자리 표시
- **Confirm Modal:** 봇 중지/삭제 등 중요 액션 전 확인
- **Empty State:** 봇/계좌 없을 때 빈 화면 안내 메시지

## 15. UX 정책

### 15.1 봇 상태 표시 색상

| 상태 | 색상 | 아이콘 |
|------|------|--------|
| 실행 중 (RUNNING) | 초록 | 🟢 |
| 대기 중 (PENDING) | 노랑 | 🟡 |
| 중지됨 (STOPPED) | 회색 | ⚪ |
| 오류 (ERROR) | 빨강 | 🔴 |
| 완료 (COMPLETED) | 파랑 | 🔵 |

### 15.2 중요 액션 확인 정책

아래 액션은 반드시 **Confirm Modal**로 재확인 후 실행합니다.

- 봇 중지 (운영 중 포지션이 있을 경우 포지션 처리 방식 선택)
- 봇 삭제
- 계좌 연동 삭제
- 회원 탈퇴

### 15.3 반응형 디자인

- 데스크톱 (1280px 이상): 사이드바 + 2~3 컬럼 레이아웃
- 태블릿 (768px~1280px): 축소 사이드바
- 모바일 (768px 미만): 하단 탭바, 1컬럼 레이아웃 (Phase 2)
